# Assemble Brief — Frontend Apps

Use this fragment when `designIntent.artifactKind === "frontend-app"`. Cite the local-catalog references appropriate to the user's category (restaurant-landing, saas-marketing, dashboard-admin, etc.). Reference web hits should be live web apps (linear.app, vercel.com, stripe.com), NOT API docs or backend tools.

## Quality bar

- Headlines name a concrete value prop, not "experience the future of X".
- Hero uses real-feeling imagery (photo / illustration / abstract gradient) — not stock-illustration.
- Cite at least one peer in the user's specific category, not just "all SaaS sites".
